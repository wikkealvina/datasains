2. Membaca, Menampilkan, dan Menulis Gambar

Membaca Gambar

f = system.file("images", "sample.png", package="EBImage")
img = readImage(f)
  • Menampilkan Gambar dengan Display

Disini gambar dibaca menggunakan fungsi “ReadImage” dengan cara mengambil nama file atau URL gambar sebagai masukannya.baiklah berikut cara menampilkan gambar menggunakan fungsi display()

Image <- readImage('D:/UIN/SEMESTER 2/Linear Algebra/uin.jpg')

img = readImage('D:/UIN/SEMESTER 2/Linear Algebra/uin.jpg')

5. Memanipulasi gambar

Dengan menggunakan array numerik, gambar akan dapat dengan mudah dimanipulasi oleh salah satu operator aritmatika dalam R. Contohnya, kita dapat menghasilkan gambar negatif hanya dengan mengurangi nilai gambar dari maksimumnya. Berikut adalah contohnya

img_neg = max(img) - img
display( img_neg )

Selanjutnya, Kita juga bisa meningkatkan kecerahan pada gambar dengan melalui penambahan,menyesuaikan kontras dengan perkalian dan menerapkan koreksi gamma melalui eksponen

img_comb = combine(
  img,
  img + 0.3,
  img * 2,
  img ^ 0.5
)

display(img_comb, all=TRUE)

Selanjutnya, Pada percobaan ini kita mencoba menggabungkan beberapa gambar kedalam satu objek gambar dengan multi bingkai.Kemudian kita juga dapat memotong dan membuat gambar thresholding dengan operasi matriks standart.

colorMode(Image) <- Color
display(Image[50:300, 2:300,])

Selanjutnya, Untuk melakukan transposisi gambar, anda dapat menggunakan fungsi transpose dalam pake R’s R (base") dengan fungsi t.hal ini juga dapat berfungsi pada gambar berwarna dan multiframe dengan cara menukar dimensi spasialnya, contohnya adalah :

img_t = transpose(img)
display( img_t )

6. Transformasi Spasial

Setelah mengetahui bagaimana jenis-jenis transformasi spasial dan transposisi, kita akan mengenal translasi,rotasi,refleksi,dan penskalaan. Translasi memindahkan bidang gambar dengan vektor dua dimensi yang ditentukan sedemikian rupa sehingga piksel yang berakhir di luar wilayah gambar akan dipotong,dan piksel yang masuk ke wilayah gambar diset ke latar belakang.

img_translate = translate(img, c(100,-50))
display(img_translate)

img_rotate = rotate(img, 30, bg.col = "white")
display(img_rotate)

Untuk menskalakan gambar ke dimensi yang diinginkan, gunakan pengubahan ukuran. Jika Anda hanya memberikan salah satu dari lebar atau tinggi, dimensi lainnya secara otomatis dihitung dengan tetap menggunakan rasio aspek asli.

img_resize = resize(img, w=256, h=256)
display(img_resize )

Fungsi flip dan flop masing-masing mencerminkan gambar di sekitar sumbu horizontal dan vertikal gambar.

img_flip = flip(img)
img_flop = flop(img)

display(combine(img_flip, img_flop), all=TRUE)

Transformasi linier spasial diimplementasikan menggunakan transformasi affine umum. Ini memetakan koordinat piksel gambar px menggunakan matriks transformasi 3x2 m. Misalnya, pemetaan tipis horizontal dapat diterapkan dengan:

m =  matrix(c(1, -.5, 128, 0, 1, 0), nrow=3, ncol=2)
img_affine = affine(img, m)
display( img_affine )

Package R (“EBImage”) mendefinisikan objek sebagai sekumpulan piksel dengan nilai integer unik yang sama. Biasanya citra yang mengandung objek merupakan hasil dari fungsi segmentasi. Objek dapat dihapus dari gambar tersebut. Secara default, setelah penghapusan objek, semua objek yang tersisa diberi label ulang sehingga ID objek tertinggi sesuai.

objects = list(
    seq.int(from = 2, to = max(logo_label), by = 2),
    seq.int(from = 1, to = max(logo_label), by = 2)
    )
logos = combine(logo_label, logo_label)
z = rmObjects(logos, objects, reenumerate=FALSE)
display(z, all=TRUE)

10. Segmen sel

Disimpulkan vignette ini dengan menerapkan fungsi yang dijelaskan sebelumnya untuk tugas segmentasi sel. Tujuannya adalah untuk mengidentifikasi secara komputasi dan mengkarakterisasi secara kualitatif sel dalam sampel gambar mikroskop. Pendekatan ini dapat diterapkan pada koleksi yang berisi ribuan gambar. Dimulai dengan memuat gambar inti dan badan sel. Untuk memvisualisasikan sel, dilapisi gambar-gambar ini sebagai saluran hijau dan biru dari gambar berwarna palsu.

nuc = readImage(system.file('images', 'nuclei.tif', package='EBImage'))
cel = readImage(system.file('images', 'cells.tif', package='EBImage'))

cells = rgbImage(green=1.5*cel, blue=nuc)
display(cells, all = TRUE)

nmask = thresh(nuc, w=10, h=10, offset=0.05)
nmask = opening(nmask, makeBrush(5, shape='disc'))
nmask = fillHull(nmask)
nmask = bwlabel(nmask)

display(nmask, all=TRUE)

ctmask = opening(cel>0.1, makeBrush(5, shape='disc'))
cmask = propagate(cel, seeds=nmask, mask=ctmask)

display(ctmask, all=TRUE)

segmented = paintObjects(cmask, cells, col='#ff00ff')
segmented = paintObjects(nmask, segmented, col='#ffff00')

display(segmented, all=TRUE)

11. Seni, Pemprosesan gambar dan Representasi spektral sinyal

mencoba mengintrogasi gambar dibawah ini:

library(EBImage)

image <- readImage("D:/UIN/SEMESTER 2/Linear Algebra/uin.jpg")

display(image, method = "raster", all = TRUE)

# Intensitas Histogram
hist(image)

# Generate negative image
img_neg <- max(image) - image
display( img_neg )

# dim: 633 1000    3

# crop and threshold
img_crop <- image  
img_thresh <- img_crop > 0.55
display(img_thresh)

# High-pass filtering Laplace Filter/Edge detection
fhi <- matrix(1, nrow = 3, ncol = 3)
fhi[2, 2] <- -7
img_fhi <- filter2(image, fhi)
display(img_fhi)

# Segmentation
grayimage <- channel(image,"gray")
threshold <- otsu(grayimage)
CC_th <- combine( mapply(function(frame, th) frame > th, 
                         getFrames(image), 
                         threshold, SIMPLIFY=FALSE) )
display(CC_th, all=TRUE)

disc <- makeBrush(51, "disc")
disc <- disc/sum(disc)
offset <- 0.0001
CC_bg <- filter2(image, disc)
CC_th <- image < CC_bg + offset
display(CC_th, all=TRUE)

# Watershed segmentation
nmask <- watershed(distmap(CC_th), tolerance=5, ext=20)
display(colorLabels(nmask), all=TRUE)

# Voronoi image tesselation
voronoiExamp <- propagate(x = nmask, seeds = nmask, lambda = 0.001)
voronoiPaint <- colorLabels (voronoiExamp)
display(voronoiPaint, all=TRUE)

ctmask = opening(img_fhi>0.1, makeBrush(5, shape='disc'))
cmask = propagate(img_fhi, seeds=nmask, mask=ctmask)
display(ctmask, all=TRUE)

segmented = paintObjects(channel(cmask,"gray"), image, col='#ff00ff')
segmented = paintObjects(channel(ctmask,"gray"), segmented, col='#ffff00')
display(segmented, all=TRUE)
